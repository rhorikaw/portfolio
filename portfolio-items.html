PR Hoff

David Hasselhoff as a PR Bot

A Slack bot with a David Hasselhoff theme that reminds users to close open pull requests in the team's Github repository.

This was my final project that I worked on during my time at Experian. It was an amazing opportunity to reinforce my knowledge of RESTful APIs, as I worked with both the Slack and Github APIs to assign and send pull requests to appropriate members of the team via Slack.

Due to this project being uploaded and deployed on private Experian accounts, there will not be a github repo or a live site. Sorry for the inconvenience!
